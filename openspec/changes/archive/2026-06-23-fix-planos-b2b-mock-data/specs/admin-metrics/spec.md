## ADDED Requirements

### Requirement: Exibir métricas e gráficos reais
O sistema DEVE permitir que a tela de Dashboard e Métricas do Admin mostre dados baseados nas compras e matrículas armazenadas no banco de dados.

#### Scenario: Visualizar painel inicial
- **WHEN** o administrador logar e visualizar a seção Dashboard
- **THEN** o sistema consome e exibe a receita mensal real e a performance de cursos baseada em matrículas do banco.
